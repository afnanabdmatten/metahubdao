# MetaHubDAO Smart Contract

## Overview
MetaHubDAO is a comprehensive Clarity smart contract that combines multiple DeFi and DAO functionalities into a single hub. This contract provides NFT management, DAO governance, staking mechanisms, crowdfunding capabilities, subscription services, lending facilities, and a reputation system.

## Features

### NFT Management
- Mint NFTs with custom metadata
- Track NFT ownership
- Simplified NFT registry system

### DAO Governance
- Create and manage proposals
- Voting mechanism with for/against options
- Proposal execution system
- Treasury management

### Staking System
- Stake STX tokens
- Track staking duration
- Unstake with rewards

### Crowdfunding
- Create funding projects
- Track project milestones
- Manage backer contributions
- Project state management

### Subscription Services
- Create subscription-based services
- Manage subscription durations
- Track user subscriptions

### Lending Platform
- Collateralized lending
- Loan management
- Repayment tracking

### Reputation System
- Dynamic reputation scoring
- User reputation tracking
- Owner-controlled reputation awards

## Contract Functions

### NFT Functions
- `mint-nft`: Mint new NFTs with metadata

### Crowdfunding Functions
- `create-project`: Initialize new funding projects
- `fund-project`: Contribute to existing projects

### Staking Functions
- `stake`: Stake STX tokens
- `unstake`: Withdraw staked tokens

### Subscription Functions
- `subscribe`: Subscribe to services

### Lending Functions
- `borrow`: Take out collateralized loans
- `repay`: Repay existing loans

### Governance Functions
- `create-proposal`: Create new proposals
- `vote`: Vote on active proposals
- `execute-proposal`: Execute passed proposals

### Reputation Functions
- `add-reputation`: Add reputation points
- `get-reputation`: Query reputation scores

## Error Codes
- `u100-u199`: Crowdfunding errors
- `u200-u299`: Staking errors
- `u300-u399`: Lending errors
- `u400-u499`: Governance errors
- `u500-u599`: Reputation errors
- `u600-u699`: NFT errors
- `u700-u799`: Project errors
- `u800-u899`: Stake errors
- `u900-u999`: Subscription errors
- `u1000-u1099`: Lending errors
- `u1100-u1199`: Proposal errors
